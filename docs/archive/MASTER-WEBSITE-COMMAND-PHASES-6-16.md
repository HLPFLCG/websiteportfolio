## PHASE 6: CONTENT STRATEGY

### 6.1 CONTENT GUIDELINES

**Tone of Voice:**

*Brand Voice Attributes:*
- **Professional** - We communicate with expertise and authority
  - Use industry terminology appropriately
  - Demonstrate deep knowledge
  - Provide valuable insights
  - Back claims with data

- **Bold** - We're confident in our abilities and results
  - Make strong statements
  - Use powerful language
  - Showcase achievements
  - Challenge status quo

- **Transparent** - We're honest about everything
  - Clear pricing upfront
  - Honest about timelines
  - Open about process
  - Admit limitations

- **Friendly** - We're easy to work with
  - Conversational tone
  - Use contractions
  - Avoid jargon overload
  - Be approachable

- **Direct** - We get to the point
  - Short sentences
  - Clear statements
  - No fluff
  - Action-oriented

- **Empowering** - We help clients succeed
  - Focus on benefits
  - Provide solutions
  - Share knowledge
  - Enable growth

**Writing Style Guidelines:**

*Sentence Structure:*
- Average length: 15-20 words
- Mix short (5-10 words) and medium (15-25 words)
- Avoid long sentences (30+ words)
- Use active voice 90% of the time
- One idea per sentence

*Paragraph Structure:*
- Length: 3-4 sentences maximum
- One main idea per paragraph
- Use transition words
- White space between paragraphs
- Mobile-friendly length

*Word Choice:*
- Use simple words when possible
- Explain technical terms
- Avoid jargon unless necessary
- Use power words for impact
- Be specific, not vague

*Grammar and Style:*
- Use contractions (we're, you'll, it's)
- Use second person (you, your)
- Use present tense when possible
- Use numbers instead of words (5 not five)
- Use bullet points for lists

**Content Types:**

*Homepage Copy:*
```
Headline: [Action verb] + [Benefit] + [Timeframe/Qualifier]
Example: "Build Your Dream Website in 72 Hours"

Subheadline: [Expand on benefit] + [Address pain point]
Example: "Professional web development without the 6-month wait or $50K price tag"

Body: [Problem] → [Solution] → [Proof] → [CTA]
Example: "Tired of slow agencies? We deliver premium websites in 72 hours. 
500+ projects completed with 98% satisfaction. Get started today."
```

*Service Pages:*
```
Structure:
1. What it is (1-2 sentences)
2. Who it's for (1 sentence)
3. What's included (bullet points)
4. How it works (3-4 steps)
5. Why choose us (3-4 reasons)
6. Results/proof (stats, testimonials)
7. CTA (clear next step)
```

*About Page:*
```
Structure:
1. Mission statement (1 sentence)
2. Origin story (2-3 paragraphs)
3. Values (3-5 core values)
4. Team (photos + bios)
5. Achievements (stats, awards)
6. CTA (join us, work with us)
```

*Blog Posts:*
```
Structure:
1. Compelling headline (8-12 words)
2. Hook (first paragraph grabs attention)
3. Problem statement
4. Solution/insights (main content)
5. Examples/case studies
6. Actionable takeaways
7. CTA (related service, newsletter)

Length: 1,500-2,500 words
Readability: 8th grade level
Formatting: H2/H3 headings, short paragraphs, bullet points
```

### 7.1 MOBILE-FIRST DESIGN PRINCIPLES

**Core Principles:**

1. **Content Priority**
   - Most important content first
   - Progressive disclosure
   - Collapsible sections
   - Simplified navigation

2. **Touch-Friendly**
   - Minimum 44x44px touch targets
   - Adequate spacing between elements
   - Large, easy-to-tap buttons
   - Swipe gestures where appropriate

3. **Performance**
   - Smaller images
   - Fewer resources
   - Lazy loading
   - Simplified animations

4. **Readability**
   - Larger font sizes
   - Shorter line lengths
   - More white space
   - Clear hierarchy

### 7.3 MOBILE TOUCH INTERACTIONS

**Touch Gestures:**

```javascript
// Swipe to navigate
let touchStartX = 0;
let touchEndX = 0;

document.addEventListener('touchstart', e => {
  touchStartX = e.changedTouches[0].screenX;
});

document.addEventListener('touchend', e => {
  touchEndX = e.changedTouches[0].screenX;
  handleSwipe();
});

function handleSwipe() {
  const swipeThreshold = 50;
  const diff = touchStartX - touchEndX;
  
  if (Math.abs(diff) > swipeThreshold) {
    if (diff > 0) {
      // Swiped left - next page/section
      navigateNext();
    } else {
      // Swiped right - previous page/section
      navigatePrevious();
    }
  }
}

// Pull to refresh
let touchStartY = 0;
let touchEndY = 0;

document.addEventListener('touchstart', e => {
  touchStartY = e.changedTouches[0].screenY;
});

document.addEventListener('touchmove', e => {
  touchEndY = e.changedTouches[0].screenY;
  
  if (window.scrollY === 0 && touchEndY > touchStartY + 100) {
    // Show refresh indicator
    document.body.classList.add('pull-to-refresh');
  }
});

document.addEventListener('touchend', e => {
  if (document.body.classList.contains('pull-to-refresh')) {
    // Refresh page
    location.reload();
  }
  document.body.classList.remove('pull-to-refresh');
});

// Tap to show/hide elements
document.querySelectorAll('.tap-to-reveal').forEach(element => {
  element.addEventListener('touchstart', function() {
    this.classList.toggle('revealed');
  });
});
```

### 7.4 MOBILE PERFORMANCE

**Mobile-Specific Optimizations:**

```javascript
// Detect mobile device
const isMobile = /iPhone|iPad|iPod|Android/i.test(navigator.userAgent);

if (isMobile) {
  // Disable hover effects
  document.body.classList.add('mobile-device');
  
  // Reduce animation complexity
  document.documentElement.style.setProperty('--animation-duration', '0.2s');
  
  // Lazy load more aggressively
  const images = document.querySelectorAll('img');
  images.forEach(img => {
    img.loading = 'lazy';
  });
  
  // Disable parallax effects
  document.querySelectorAll('.parallax').forEach(el => {
    el.style.transform = 'none';
  });
  
  // Simplify particles
  if (typeof particlesJS !== 'undefined') {
    particlesJS('particles-js', {
      particles: {
        number: { value: 30 }, // Reduced from 80
        size: { value: 2 }, // Smaller
        move: { speed: 1 } // Slower
      }
    });
  }
}
```

        btn.classList.add('active');
        
        const filter = btn.getAttribute('data-filter');
        
        portfolioItems.forEach(item => {
          if (filter === 'all' || item.getAttribute('data-category').includes(filter)) {
            item.style.display = 'block';
            item.classList.add('fade-in');
          } else {
            item.style.display = 'none';
          }
        });
      });
    });
    
    // Search functionality
    const searchInput = document.getElementById('portfolio-search');
    searchInput.addEventListener('input', (e) => {
      const searchTerm = e.target.value.toLowerCase();
      
      portfolioItems.forEach(item => {
        const title = item.querySelector('.portfolio-title').textContent.toLowerCase();
        const excerpt = item.querySelector('.portfolio-excerpt').textContent.toLowerCase();
        
        if (title.includes(searchTerm) || excerpt.includes(searchTerm)) {
          item.style.display = 'block';
        } else {
          item.style.display = 'none';
        }
      });
    });
