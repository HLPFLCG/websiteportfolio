## üîß Automatic Deployment

### How It Works
Every time you push to the `main` branch, GitHub Actions automatically:
1. Checks out the latest code
2. Sets up GitHub Pages
3. Uploads the website files
4. Deploys to GitHub Pages
5. Makes it live at https://hlpflcg.github.io/websiteportfolio/

### Workflow File Location
`.github/workflows/deploy.yml`

### Manual Deployment
You can also trigger deployment manually:
1. Go to: https://github.com/HLPFLCG/websiteportfolio/actions
2. Click "Deploy to GitHub Pages"
3. Click "Run workflow"

## üéØ Custom Domain Setup (Optional)

To use a custom domain like `hlpfl.dev`:

### Step 1: Configure DNS
Add these DNS records at your domain registrar:

```
Type: CNAME
Name: www
Value: hlpflcg.github.io
```

```
Type: A
Name: @
Value: 185.199.108.153
Value: 185.199.109.153
Value: 185.199.110.153
Value: 185.199.111.153
```

### Step 2: Configure GitHub Pages
1. Go to: https://github.com/HLPFLCG/websiteportfolio/settings/pages
2. Under "Custom domain", enter: `hlpfl.dev` or `www.hlpfl.dev`
3. Check "Enforce HTTPS"
4. Save

### Step 3: Update Repository
Create a file named `CNAME` in the root with your domain:
```
hlpfl.dev
```
